Class

CustomShaderMaterial

CustomShaderMaterial(options)

This class lets you use your own custom Vertex Shaders along with predefined ThreeJS Fragmet Shaders. This takes away the hastle of writing code for lighting and shaing.
Constructor

# new CustomShaderMaterial(options)

Creates an instance of the CustomShaderMaterial class. The variable newPos and newNormal must be defined in the main section of the indjected shader.
Parameters:
Name Type Attributes Description
options Object Options for material.
baseMaterial string Base Material. The material whos fragment shader is used. Any type from the exported TYPES object
vShader CustomShader Custom Vertex Shader
fShader CustomShader <optional>
 Custom Fragment Shader
uniforms Object <optional>
 Custom Uniforms to be passed to the shader.
passthrough Object <optional>
 Any custom options to be passed to the underlying base material.

Example
const material = new CustomShaderMaterial({
    baseMaterial: TYPES.PHYSICAL,
    vShader: {
        defines: vertex.defines,
        header: vertex.header,
        main: vertex.main,
    },
   fShader: {
        defines: fragment.defines,
        header: fragment.header,
        main: fragment.main,
    },
    uniforms: {
        uTime: { value: 1.0 },
        uResolution: { value: new THREE.Vector3() },
    },
    passthrough: {
        flatShading: true,
    },
});
material.uniforms.uResolution.set(1920, 1080, 0);
